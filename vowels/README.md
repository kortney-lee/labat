# Vowels.org MVP

Static-first Next.js TypeScript publication platform for Vowels.org.

## Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- Node.js runtime
- Vercel hosting target

## Run locally
1. `npm install`
2. `npm run dev`
3. open `http://localhost:3000`

## Required environment variables
- `NEXT_PUBLIC_SITE_URL=https://vowels.org`
- `NEXT_PUBLIC_GA_MEASUREMENT_ID=G-XXXXXXXXXX` (optional)

## MVP includes
- Homepage
- Article pages
- Category pages
- Search
- Newsletter signup
- Ad slots
- Sponsored labels
- RSS feed (`/rss.xml`)
- Sitemap (`/sitemap.xml`)
- News sitemap (`/news-sitemap.xml`)
- Article schema
- Editorial policy
- Health disclaimer
- Privacy policy
- Terms
